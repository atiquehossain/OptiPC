## 🎯 **How to Use**

### **Step 1: Download**
Click the download link that matches your computer type above.

### **Step 2: Extract**
1. Save the ZIP file to your Desktop or Downloads
2. Right-click the ZIP file
3. Select "Extract All..." or "Extract here"
4. Open the extracted folder

### **Step 3: Run**
1. Double-click `OptiPC_[version].exe`
2. If Windows shows a security warning, click "More info" → "Run anyway"
3. **Enjoy OptiPC!** No installation needed!

## ❓ **Need Help?**

**Not sure which version to download?**
- Press `Windows Key + R` → Type `dxdiag` → Enter
- Look for "System Type" line
- If it says "64-bit", download the x64 version
- If it says "32-bit", download the x86 version

**Having trouble running?**
- Check the [TROUBLESHOOTING_FRIEND.md](TROUBLESHOOTING_FRIEND.md) file
- Make sure Windows is up to date
- Try running as administrator (right-click → "Run as administrator")
